Bad code smell adalah indikasi atau tanda dalam source code yang menunjukkan adanya masalah yang lebih dalam pada desain atau kualitas code. Code smell bukan merupakan bug atau error, melainkan pelanggaran terhadap prinsip-prinsip desain yang fundamental, yang dapat berdampak negatif pada kualitas desain secara keseluruhan.

Meskipun code smell tidak mencegah program untuk berfungsi dengan baik, namun keberadaannya mengindikasikan adanya kelemahan desain yang dapat memperlambat proses pengembangan atau meningkatkan risiko munculnya bug di masa depan. Code smell juga dapat menjadi indikator dari faktor-faktor yang berkontribusi pada technical debt dalam pengembangan perangkat lunak.

Bad code smell bukan merupakan masalah yang langsung terlihat, tetapi merupakan sinyal adanya potensi masalah yang lebih dalam pada desain dan kualitas code. Mengidentifikasi dan mengatasi code smell sejak dini dapat membantu menjaga kualitas code dan menghindari masalah yang lebih besar di kemudian hari dalam proses pengembangan perangkat lunak.

Dispensables

Dispensable adalah sesuatu yang tidak berguna dan tidak diperlukan didalam sebuah code. Keberadaannya tidak akan membuat code menjadi lebih bersih, lebih efisien, dan lebih mudah dipahami.

Change Preventers

Change preventers adalah smell yang mengartikan bahwa jika Anda perlu mengubah sesuatu dalam code anda, maka Anda harus membuat banyak perubahan di tempat lain juga. Pengembangan program jauh lebih rumit, sulit serta biayanya jauh lebih mahal sebagai hasilnya.
